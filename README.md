ProtoToObjcTransition
=====================

make .proto to objc protocolbuffer


Usage
=====

- 打开终端输入命令，进入ProtocolBuffers文件夹

'''linux
cd ProtoToObjcTransition/ProtocolBuffers-ObjC
./autogen.sh
./configure
sudo make install
'''

- 进入Protobuf_Generator文件夹，使用文本编辑run
